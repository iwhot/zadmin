# zadmin
go写的一个小博客


#### 编译运行
```
windows:
1、git clone https://github.com/iwhot/zadmin.git
2、cd zadmin
3、go build -o zadmin.exe ./cmd/zadmin/main.go
4、./zadmin.exe

linux:
1、git clone https://github.com/iwhot/zadmin.git
2、cd zadmin
3、go build -o zadmin ./cmd/zadmin/main.go
4、./zadmin
```
